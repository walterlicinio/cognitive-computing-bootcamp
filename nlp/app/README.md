#<center> Wordcloud DataApp

## Configuração

1. Instalar depêndencias:
    - No terminal, executar os seguintes comandos:
        ```shell
        pip install -r requiriments.txt 
        ```
        ```shell
        python config.py
        ```

2. Execução:
    - No diretório `nlp/app/`, executar o seguinte comando:
        ```shell
        sh app.sh
        ```
